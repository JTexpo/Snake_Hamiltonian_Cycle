How to use this app
Upon entering the webpage, you will notice the classic Snake game running. There are five buttons available: RESET, Spike, Apple, Erase, and "Show Snake Moves."

  • The RESET button will reset the game entirely.
  • The Spike button allows you to place a spike in the snake's path, change the snakes route.
  • The Apple button will place an apple for the snake to eat, which will increase its size.
  • The Erase button removes a spike or apple from the game.
  • The "Show Snake Moves" button will display a second canvas below, showing the Hamiltonian cycle that the snake is following.

What is a Hamiltonian Cycle?
A Hamiltonian cycle is a path in a graph that visits every node exactly once.

In the game Snake, a Hamiltonian cycle can help determine the most optimal path for the snake to follow without colliding with its own body. This cycle is NP-Hard, which means that it can take an impractically long time to find a solution. Thus, the game is limited to a 6x6 grid in order to provide the most fluid gameplay.

The snake will dynamically generate a new Hamiltonian cycle whenever a spike is placed or removed. A depth first search will be used to find the longest Hamiltonian cycle, making a circut which connects the snakes head to it's tail.
